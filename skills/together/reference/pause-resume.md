# Pause and Resume

If work cannot continue safely, write a structured resume report.

## Resume report fields

- objective
- current decomposition
- completed work
- in-flight step at pause time
- attempted routes
- failures encountered
- known limits or wait states
- next recommended batch
- exact commands or prompts to resume
- unresolved risks

Store the report both:
- in chat summary form
- in a file when possible

The file form is the canonical handoff record.
