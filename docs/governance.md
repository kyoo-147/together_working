# Work Governance

Together assumes workers need boundaries.

Rule:
- worker does assigned work only
- review checks output quality
- verification checks scope and contract
- Codex decides merge

Governance flow:

```text
Task
↓
Assignment
↓
Execution
↓
Review
↓
Verification
↓
Merge Decision
```

Codex role:
- planner
- coordinator
- verifier
- integrator
- merge authority
