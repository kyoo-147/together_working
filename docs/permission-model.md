# Permission Model

Together separates work by role.

Roles:
- Observer
- Researcher
- Implementer
- Reviewer
- Integrator

Role intent:
- Observer: read, search, analyze
- Researcher: gather, compare, summarize
- Implementer: modify assigned scope only
- Reviewer: review, approve, reject
- Integrator: merge and final decision

Default:
- Codex acts as Integrator
