# Approval Gate

Every real external execution batch must be approved by the user first.

## Batch contents

- objective
- capability
- recommended agent
- recommended model
- fallback chain
- scope boundaries
- expected output schema
- main risks

## Minimum ask

Keep the approval ask short and concrete:

"Proposed batch:
- capability: ...
- worker: ...
- model: ...
- fallback: ...
- output: ...
Approve this batch?"

Do not hide dispatch details.
